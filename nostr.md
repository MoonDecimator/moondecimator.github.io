# Nostr

This page explains my public Nostr identity and the purpose of NIP-05 verification on this site.

Nostr is an open and decentralized protocol for social communication. Identity is based on public and private key cryptography rather than on an account controlled by a single platform.

This site is also used to publish the file `/.well-known/nostr.json`, which allows compatible Nostr clients to verify that this domain is associated with my public key.

If you are here specifically for verification, the key resource is:

`/.well-known/nostr.json`
